JwtMiddleware

Methods

(inner) checkJWT(req, res, next) → {void}

Description:
  • Middleware to check if the user has a valid JWT (JSON Web Token). This middleware checks the `authToken` from the user's cookies. If the token is valid, the decoded user information is attached to the `req` object. If the token is invalid or missing, it returns a 401 Unauthorized error.
Source:
  • middlewares/private.js, line 9
Example
// Usage in a route (with the private middleware namespace)
const private = require('../middlewares/private');

app.get('/protected', private.checkJWT, (req, res) => {
  res.json({ message: 'This is a protected route', user: req.user });
});
